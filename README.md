## Development Setup

Setting up development and installing dependencies. (Using python virtual environments)

### Windows

```bash
python.exe -m venv .venv
./.venv/Scripts/activate
pip install -r requirements.txt
```

### Linux / Mac

```bash
python3 -m venv .venv
source ./.venv/bin/activate
pip install -r requirements.txt
```
